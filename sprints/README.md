# Спринты

Папка для планирования и отслеживания спринтов разработки PPT PowerKeys.

## Структура спринта

Каждый спринт — отдельная подпапка:

```
sprints/
├── sprint-01-mvp/
│   ├── goals.md          # Цели спринта
│   ├── backlog.md        # Задачи
│   └── retrospective.md  # Итоги
└── sprint-02-...
```
