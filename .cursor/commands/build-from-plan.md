# /build-from-plan

Реализуй код по готовому плану архитектора.

1. Возьми сохранённый план (из workspace) или укажи путь к нему.
2. Запусти субагента `builder`.
3. Builder создаёт ветку `cursor/<кратко>-7151`, пишет код по плану и конвенциям `.cursor/rules/`.
4. Коммиты по Conventional Commits, затем PR в `main` по шаблону с Sprint и Task ID.
5. Отметь, что требуется ручная проверка в PowerPoint (Windows), если сборку нельзя выполнить здесь.
