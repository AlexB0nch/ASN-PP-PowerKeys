# 0002. Стек: VSTO + C# + Office Interop

- **Статус:** Accepted
- **Дата:** 2026-06-26

## Контекст

Нужен плагин для PowerPoint (Windows, 2013–2021, M365) с лентой, глобальными шорткатами и доступом к объектной модели слайдов.

## Решение

Visual Studio Tools for Office (VSTO) на C#, интеграция через `Microsoft.Office.Interop.PowerPoint`, регистрация COM Add-in в `HKCU` (per-user), установка через `.msi`/`.exe`.

## Альтернативы

- **Office JS Add-ins** — кроссплатформенно, но ограниченный доступ к объектной модели и нет глобальных системных шорткатов.
- **VBA** — не распространяется как продукт, слабый UI/настройки.

## Последствия

- Полный доступ к COM-модели и Ribbon.
- Привязка к Windows + .NET; сборка/тест только на Windows с установленным Office.
- CI/cloud-агенты на Linux не могут собрать/протестировать против Office (см. ADR 0003).
