# PPT PowerKeys — Плагин для Microsoft PowerPoint

> Плагин для PowerPoint, воспроизводящий и расширяющий функционал **ShortCut Tools** — незаменимого инструмента для консультантов McKinsey, BCG, Bain и всех, кто работает со слайдами профессионально.

---

## Общая концепция

**PPT PowerKeys** — это COM-надстройка (`.dll`) для Microsoft PowerPoint (Windows, версии 2013–2021, Microsoft 365), которая добавляет в ленту отдельную вкладку **PowerKeys** и регистрирует глобальные клавиатурные сочетания для >100 команд. Все шорткаты настраиваемы через встроенный менеджер горячих клавиш.

**Технический стек (legacy VSTO):**
- **Visual Studio 2022** + workload VSTO (Visual Studio Tools for Office) на C# — не VS Code и не VS 2019
- Интеграция через `Microsoft.Office.Interop.PowerPoint`
- Регистрация COM Add-in через реестр Windows
- Установка через `.msi` / `.exe` инсталлятор

## Категории команд

### 1. 🔲 Выравнивание (Alignment)

Все команды выравнивают объекты **относительно последнего выделенного** (не края слайда) — поведение, которого нет в стандартном PowerPoint.

| Команда | Шорткат | Описание |
|---|---|---|
| Выровнять по левому краю | `Alt + 1` | Левые края всех объектов → левый край последнего выбранного |
| Выровнять по центру (горизонт.) | `Alt + 2` | Центры по горизонтали |
| Выровнять по правому краю | `Alt + 3` | Правые края → правый край последнего выбранного |
| Распределить по горизонтали | `Alt + 4` | Равномерный интервал по горизонтали |
| Выровнять по верхнему краю | `Alt + F1` | Верхние края → верх последнего выбранного |
| Выровнять по высоте (центр) | `Alt + F2` | Центры по вертикали |
| Выровнять по нижнему краю | `Alt + F3` | Нижние края → низ последнего выбранного |
| Распределить по вертикали | `Alt + F4` | Равномерный интервал по вертикали |
| Скопировать и выровнять влево | `Alt + Shift + 1` | Дублирует и выравнивает по левому краю |
| Скопировать и выровнять вправо | `Alt + Shift + 3` | Дублирует и выравнивает по правому краю |
| Скопировать и выровнять вверх | `Alt + Shift + F1` | Дублирует и выравнивает по верхнему краю |
| Скопировать и выровнять вниз | `Alt + Shift + F3` | Дублирует и выравнивает по нижнему краю |
| Выровнять левый край к правому | `Ctrl + Shift + 1` | Левый край объекта → правый край последнего выбранного |
| Выровнять правый край к левому | `Ctrl + Shift + 3` | Стыкует объекты справа налево |
| Выровнять верх к низу | `Ctrl + Shift + F1` | Стыкует объекты сверху вниз |
| Выровнять низ к верху | `Ctrl + Shift + F3` | Стыкует объекты снизу вверх |
| Скопировать позицию объекта | `User defined` | Сохраняет координаты X/Y выбранного объекта в буфер |
| Вставить позицию объекта | `User defined` | Применяет сохранённые координаты к выбранному объекту |

## Встроенная вкладка PowerKeys (Ribbon)

Вкладка в ленте PowerPoint разделена на логические группы:

```
[PowerKeys] 
  ├── [Выравнивание] — кнопки Align Left/Center/Right/Top/Middle/Bottom/DistH/DistV
  ├── [Размер]       — кнопки Same Width/Height, Resize to Edge
  ├── [Объекты]      — Insert: Rectangle, Square, Circle, Line, Textbox, Arrow
  ├── [Дублировать]  — Duplicate: Right, Left, Down, Up
  ├── [Порядок]      — To Front, To Back, Forward, Backward, Group, Ungroup
  ├── [Формат]       — Fill Color, Line Color, Text Color, Format Painter
  ├── [Текст]        — Paste Unformatted, Addup, Superscript, Subscript
  ├── [Слайды]       — Zoom Toggle, Slide Sorter, Print, Copy Slide
  └── [Настройки]    — Shortcut Manager, Color Scheme, Reset to Defaults
```

## Дополнительные инновационные функции

_(Не в ShortcutTools, но логично добавить)_

### Smart Color Picker
- Отображает палитру Slide Master при нажатии `Alt+G` / `Alt+L`
- Запоминает последние 5 использованных цветов
- Поддержка пипетки (eyedropper)
- Поддержка HEX/RGB ввода

### Консультантский режим (Consulting Mode)
- Предустановленный профиль шорткатов McKinsey/BCG
- Автоматическое выравнивание по сетке 0.1 cm
- Snap to nearest object при перемещении

### Multi-slide Paste
- Вставка объекта сразу на несколько выбранных слайдов (`User defined`)
- Удаление объекта со всех выбранных слайдов

### Object Statistics
- `Alt + A` на числовых полях суммирует значения и показывает результат в статус-баре
- Поддержка MIN / MAX / AVG помимо SUM

### Smart Duplicate
- `Alt + D` дублирует с тем же шагом, что и предыдущее дублирование (как Ctrl+D в Figma)
- Запоминает последний использованный вектор смещения

### Slide Backup Manager
- Выделенные слайды перемещаются в специальный раздел "Backup" в конце файла
- Быстрое скрытие/показ всего раздела Backup
